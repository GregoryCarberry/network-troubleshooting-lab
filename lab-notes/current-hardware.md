# Current Hardware State

This lab uses a mix of enterprise and SMB hardware to maximise learning value.

## Core Routing

- BT Home Hub 5A (OpenWrt)
  - Single Layer 3 authority
  - NAT / Firewall / DHCP / DNS

## Switching

- Zyxel GS1920-24 (Layer 2)
- Cisco SG300-28 (Layer 2)

## Wireless

- Cisco WLC 2504 (Management VLAN 99)
- Cisco 2602i APs
- Cisco 3802i APs

## Philosophy

Older hardware is intentionally retained where viable.

The goal is not to replace equipment for novelty,
but to extract maximum learning and performance from available resources.
